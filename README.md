# WEBMATH

Online Math Calculator

## Instructions

1. Install:

```
pip install webmath
```

2. Run Programm:

```python
# import webmath and asyncio
import asyncio
from web_math import webmath

# make def
async def bread():
    output = await webmath.calculate("3+3")
    print(output)

#run def
asyncio.run(bread())
```
